# Sample JUnit Testing using NetBeans

This project shows how to create and run JUnit tests in NetBeans IDE.  
It includes sample unit tests for:

- VerifyUserLogin  
- PayrollCalculator  

Both tests use white box testing logic and make sure all methods work as expected.

## Step 1
 Open the project

- Extract the LoginModule.zip
- Right click the ZIP.
- Extract here.
- Dapat lalabas yung folder na LoginModule.
 

## Step 2
 Open NetBeans 8.2 (Di gagana yung ibang version ng NetBeans)
 Open the project

-Go to File
- Click Open Project
- Hanapin yung extracted zip na LoginModule
- Click Open Project

Dapat makikita na yung project sa left side na bar

## Step 3
 Create Unit Test class
 Gagawin mo to sa dalawang class mamaya

- On the left side (Projects window), Hanapin mo yung VerifyUserLogins.java
- RIGHT CLICK
- Click Tools
- Click Create/Update Tests
- Select JUnit 4
- Check all checkboxes.
- Click OK

Dapat ganto maging itsura ng directory mo

Test Packages
    loginmodule
        VerifyUserLoginTest.java

## Step 4
  Dito Kana gagawa ng test code. Kahit ichat gpt mo nalang to

- Click VerifyUserLoginTest.java

- Create test sa PayrollCalculator.java
- Right click PayrollCalculator.java → Tools → Create/Update Tests → OK
- Open PayrollCalculatorTest.java tas gawa kana test code

## Step 5
 Run the Test File

- Right click VerifyUserLoginTest.java
- Click Test File

- Right click PayrollCalculatorTest.java
- Click Test File

Kelangan nakalagay is PASSED (Green)
