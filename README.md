# Sample JUnit Testing using NetBeans

This project shows how to create and run JUnit tests in NetBeans IDE.  
It includes sample unit tests for:

- VerifyUserLogin  
- PayrollCalculator  

Both tests use white box testing logic and make sure all methods work as expected.

## Features
- Java project using NetBeans 8.2  
- JUnit 4 test cases  
- Simple payroll computations  
- Login validation example  
- All unit tests pass  

## How to Run
1. Open the project in NetBeans  
2. Right click any test file  
3. Select Test File  
4. Check the Test Results window  

## Notes
- Install JUnit plugin in NetBeans if missing  
- Clean and Build before running tests  
- assertEquals is used to compare expected vs actual values
